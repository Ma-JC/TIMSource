Welcome to SNVIO!

SNVIO is a user-friendly database dedicated to provide the results of pan-cancer immunogenomic analyses based on next-generation sequencing data. Sources of the databases include TCGA, CPTAC, and multiple published WES-based immunotherapeutic cohorts.

This module provides the short summary of the gene mutation landscape and its prognostic value in one specific cancer type. Specifically, this module enables the integrative immunogenomic analyses of the correlations of gene mutation and tumor microenvironment characteristics.

CPTAC Database

  • Infiltrating immune cells
  • Immune-related signatures
  • Differential protein analysis
  • Survival analysis

This module can be used to assess the gene mutation and proteomics-based immune features. Because the proteomic layer reflects much more accurately the cellular function, this module will aid the exploration of the cellular mechanisms of regulating tumor microenvironment.

Ref_ICI datasets

  • Survival analysis
  • Immunetherapy response
  • Tumor mutation burden

This module describes the association between gene mutation and immunotherapeutic outcomes in published clinical studies of immune checkpoint inhibitors.
